+ 借助DS解析需求，大致确定项目结构以及技术栈并创建项目

+ VSC接入DS替代copilot

# Day 2

+ 基于视频参考，将原生 JavaScript 项目的登录/注册模块迁移至 React，并优化 UI/UX
+ Next.js 中，组件默认为服务端组件（Server Component）,无法使用 React Hooks 或交互事件。需要在文件顶部添加 'use client' 指令，才能将其显式声明为客户端组件（Client Component）

# Day 3

+ 本次开发收获：

  1. 优化登录/注册模块切换逻辑
     - 使用 CSS transition 实现平滑切换动画
     - 通过状态管理控制面板显示/隐藏

  2. 借助 AI 实现背景图鼠标跟随浮动效果
     - 基于 requestAnimationFrame 实现物理惯性动画
     - 封装为可复用的 useMouseFollower Hook

  3. 深入理解 object-fit 属性
     - 核心要点：img 元素必须有明确的宽度和高度
     - 配合 position: absolute + inset: 0 可完美铺满父容器

# Day 4

**1. 表单处理与验证**

- **zod**：用 `z.object`、`.email()`、`.min()`、`.refine()` 处理表单验证规则
- **react-hook-form**：通过 `register`、`handleSubmit`、`errors`、`isSubmitting` 管理表单状态，处理表单提交逻辑
- **zodResolver**：连接 zod 与 react-hook-form

**2. JWT 与 Cookie 存储**

- 用 `jwt.sign({ userId, email }, SECRET, { expiresIn: "7d" })` 生成 token
- 存入 httpOnly Cookie并配置

**3. Next.js 中间件路由保护**

- 了解Next中间件工作流程
- 用 `NextRequest` 获取请求信息，`NextResponse` 控制响应，在请求到达页面前统一完成鉴权。
