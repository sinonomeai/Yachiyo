## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Next.js 16 (App Router) |
| UI | React 19 + Tailwind CSS 4 |
| 表单 | react-hook-form + zod |
| 状态管理 | Zustand (客户端状态) + React Query (服务端状态) |
| AI | @ai-sdk/react + @ai-sdk/deepseek |
| 动画 | GSAP |
| 数据库 | PostgreSQL + pgvector (向量检索) |
| 认证 | JWT (jsonwebtoken) + httpOnly Cookie |
| UI 库 | Ant Design 6 (message 等) · react-markdown |

## 关键依赖

- `@ai-sdk/react` + `@ai-sdk/deepseek` — AI 流式对话与深度思考
- `react-markdown` + `remark-gfm` + `rehype-highlight` — Markdown 渲染
- `@tanstack/react-query` — 服务端状态缓存与乐观更新
- `zustand` — 轻量客户端状态管理
- `gsap` — 加载动画
- `jsonwebtoken` — JWT 签发与验证
